# IBM_PC_Sim

A simple project simulating an IBM PC environment using Batch files.

## Features
* IBM PC command simulation.
* Chat integration system.

## Getting Started
### For Windows User
1. Clone this repository.
2. Run `IBM_PC.bat` to run it normally.

### For Linux and macOS Users
This project primarily uses Windows Batch (`.bat`) scripts. To run this on a non-Windows system, you will need a compatibility layer:
* **Wine:** You can install [Wine](https://www.winehq.org/) to run Windows-based `.bat` or `.exe` files on Linux or macOS.
* **Alternative:** You can manually port the logic into a Shell script (`.sh`).
